# Army Evaluation Date Calculator

This is a web-based application built with Python and Streamlit to automate the date and non-rated time calculations for Army Evaluation Reports (OERs and NCOERs). 

**Calculations are strictly in accordance with AR 623-3 and DA PAM 623-3.**

### Features:
* Calculates total calendar days for the base rating period.
* Dynamically adds and subtracts non-rated periods using standard DA PAM 623-3 codes.
* Flags overlapping non-rated dates.
* Alerts the user if the evaluation falls below the 90-day minimum threshold.
* Outputs the exact Rated Months and Rated Days required for Box 1i on DA Form 67-10 or DA Form 2166-9.
